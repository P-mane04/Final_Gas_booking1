# Online gas booking system



\# Online Gas Booking Management System



\## 🚀 Tech Stack

\- Java (JSP, Servlet)

\- MySQL8

\- HTML, CSS, JavaScript



\## 📌 Features

\- Admin: Manage distributors

\- Distributor: Add consumers

\- Booking system for gas cylinders



\## ⚙️ How to Run

1\. Import project in Eclipse

2\. Configure Apache Tomcat

3\. Setup MySQL database change password and username 

4\. Run on server
